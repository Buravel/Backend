### 서비스 소개
여행 예산을 한눈에 파악할 수 있도록,  
타인의 여행스타일과 경비를 참고할 수 있도록,  
원하는 여행지와 스타일을 예산내에서 검색할 수 있도록  
여행 예산 계획 공유에 초점을 맞춘 버라벨(Buravel)입니다.
